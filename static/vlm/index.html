VLM Text Recognition

Parallel transcription of historical document scans across three vision-language models

Models

🔵
Gemini 2.5 Pro Google AI
🟠
Qwen-VL-Max Alibaba DashScope
🟣
Pixtral Large Mistral

Setup

  1. Install dependencies: 
    pip install -r requirements.txt
  2. Copy .env.example to .env and add your three API keys: 
    GEMINI_API_KEY="..."
QWEN_API_KEY="..."
MISTRAL_API_KEY="..."
  3. Run on an image — all three models are queried in parallel: 
    python src/vlm.py data/raw/geneve_1917_p1.jpg
  4. Custom prompt: 
    python src/vlm.py image.jpg --prompt "Extract all numbers from this form."

Output

Each model's transcription is printed to stdout, separated by headers:

────────────────────────────────────────────────────────────
  GEMINI
────────────────────────────────────────────────────────────
[transcribed text]

────────────────────────────────────────────────────────────
  QWEN
────────────────────────────────────────────────────────────
[transcribed text]

────────────────────────────────────────────────────────────
  MISTRAL
────────────────────────────────────────────────────────────
[transcribed text]
